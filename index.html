‚è≥ Garis Waktu Perkembangan Model Atom

1803: John Dalton

Model Atom Bola Pejal: Atom adalah partikel kecil, keras, dan tidak dapat dibagi.

Model awal atom Dalton (bola pejal).
1897: J.J. Thomson

Model Puding Kismis (**Plum Pudding Model**): Penemuan **Elektron**. Atom sebagai massa positif dengan elektron negatif tertanam di dalamnya.

Diagram Model Puding Kismis Thomson.
1911: Ernest Rutherford

Model Atom Nuklir: Hasil dari Percobaan Lembaran Emas. Menemukan **Inti Atom** (nukleus) yang bermuatan positif di pusat atom.

Ilustrasi Percobaan Lembaran Emas dan Model Nuklir Rutherford.
1913: Niels Bohr

Model Atom Bohr: Elektron bergerak mengelilingi inti dalam **lintasan (orbit) tertentu** dengan tingkat energi kuantitas.

Visualisasi Model Atom Bohr (shell elektron).
